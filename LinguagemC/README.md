Este diretório reúne meus estudos em linguagem C, desenvolvidos durante a graduação em Análise e Desenvolvimento de Sistemas.

## 📚 Conteúdos abordados

### 🔹 Fundamentos da linguagem

* Estruturas condicionais (`if`, `switch`)
* Estruturas de repetição (`for`, `while`)
* Vetores (arrays)
* Funções básicas

### 🔹 Estruturas de dados

* filas 
* Listas
* Pilhas
* Uso de `struct`

> Observação: os conceitos de estruturas de dados foram estudados em aula, ainda sem aprofundamento prático avançado.

## 🎯 Objetivo

Desenvolver a base lógica de programação e compreender o funcionamento de estruturas fundamentais utilizando a linguagem C.

## 🚀 Observações

O aprendizado está sendo feito de forma gradual, com foco na consolidação dos conceitos básicos antes de avançar para tópicos mais complexos.

Este conteúdo faz parte da minha base de programação e será evoluído com o tempo.
