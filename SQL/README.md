# 🗄️ Estudos em SQL / PostgreSQL

Este diretório contém meus estudos iniciais em SQL e modelagem de banco de dados durante a graduação em Análise e Desenvolvimento de Sistemas.

## 📚 O que aprendi até o momento

* Criação de tabelas com `CREATE TABLE`
* Uso de `PRIMARY KEY` e `FOREIGN KEY`
* Relacionamentos entre tabelas
* Tipos de dados (`VARCHAR`, `INT`, `DATE`, `NUMERIC`)
* Restrições com `NOT NULL`, `UNIQUE` e `CHECK`
* Inserção de dados com `INSERT INTO`
* Conceitos iniciais de modelagem relacional
* Integridade referencial
* Regras de exclusão com `ON DELETE`

## 🧪 Práticas realizadas

* Sistema simples de biblioteca com empréstimos
* Relacionamento entre livros e empréstimos
* Inserção de dados para testes
* Estruturação de consultas SQL básicas

## 🎯 Objetivo

Desenvolver conhecimentos em banco de dados relacionais, lógica de modelagem e manipulação de dados utilizando PostgreSQL.

## 📌 Observações

Conteúdo desenvolvido durante aulas, com foco em aprendizado gradual e evolução contínua.
